## Project Scope

This project simulates multiple users accessing a website simultaneously using Python threads. It records response times and generates a simple performance report.

## Technologies Used

- Python
- Requests
- Threading

## Features

- Simulates multiple users
- Sends HTTP GET requests
- Measures response time
- Displays request status
- Generates a performance report

## How to Run

```bash
pip install requests
python performance_lab.py
```

## Output

The application displays:

- User ID
- HTTP Status Code
- Response Time
- Performance Summary

It also creates **performance_report.txt**.
